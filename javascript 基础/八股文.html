闭包

闭包指的一个函数和对其周围状态的引用捆绑在一起，这样的组合就是闭包，闭包就是能够读取其他函数内部变量的函数。

function init() {
    var name = "Mozilla";
    function displayName() {
        alert(name);
    }
    displayName();
}
init();
